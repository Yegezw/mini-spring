学习顺序：IOC -> MVC -> JdbcTemplate -> Mybatis -> AOP

开始之前推荐学习：[设计模式之美](https://time.geekbang.org/column/intro/100039001) [设计模式总结](https://www.cnblogs.com/lidong422339/p/17500980.html)
